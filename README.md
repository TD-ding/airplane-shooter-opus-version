# 飞机大战 · Airplane Shooter

一个用 **HTML5 Canvas + 原生 JavaScript** 实现的飞机大战小游戏，全部代码在单个 `index.html` 文件中，用浏览器直接打开即可游玩，无需任何依赖或构建步骤。

## 玩法

控制你的飞机，躲避敌机与敌方子弹，击落敌机得分！

- **方向键 / WASD**：移动飞机
- **空格 / 鼠标左键**：射击
- **移动设备**：手指拖动即可同时移动并射击

## 功能特性

- ��️ 玩家飞机移动（键盘 + 鼠标 + 触摸）
- 👾 敌机自动生成，随分数提升难度（速度、密度、射击型敌机增多）
- 🔫 玩家子弹射击 + 射击型敌机向玩家瞄准开火
- 💥 矩形碰撞检测（子弹击中敌机、敌机/子弹撞击玩家）
- 🏆 实时分数显示与本地最高分记录（localStorage）
- ✨ 爆炸粒子特效、滚动星空背景、自适应屏幕缩放

## 运行

直接用浏览器打开 `index.html` 即可，无需服务器。

```bash
# 或用任意静态服务器预览
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

| 类型 | 颜色 | 血量 | 分值 | 特点 |
|------|------|------|------|------|
| 普通敌机 | 紫色 | 1 | 10 | 仅下落 |
| 射击型敌机 | 橙红 | 3 | 30 | 会向玩家瞄准射击，带血条 |
